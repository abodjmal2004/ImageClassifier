# Cats and Dogs Classifier using SVM

## Project Overview
This academic project demonstrates binary image classification (Cats vs. Dogs) using a Support Vector Machine (SVM) algorithm. It showcases the complete machine learning pipeline, from data preprocessing to hyperparameter tuning and model evaluation.

## Project Structure
- `utils.py`: Helper functions for data loading, preprocessing, and visualization.
- `train.py`: Script to train the SVM model, find best parameters, and save the model.
- `predict.py`: Script to load the trained model and make predictions on new images.
- `notebook.ipynb`: Detailed academic notebook with explanations for each step.
- `report.pdf`: Comprehensive academic report of the methodology and results.
